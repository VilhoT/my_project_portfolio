Real-time Bus Arrival Display

Function

Electronically very simple system, but maybe one of my favourites because I actually use it whenever I leave the house. No need to pick up my phone, go to the Reittiopas app, submit my location and destination, and then see when the buses arrive. I just look at the display in the hallway (LCD with I2C, please give me work so I can justify getting a real screen) and I know exactly how many minutes until my bus arrives.

How it works

So it works by using a Raspberry Pi to run a Python script to fetch the bus data via an API, calculates the difference between the arrival time and the current time, so it knows to show me how many minutes till the bus arrives. I prefer this over seeing the time when the bus arrives because it's simpler. Then, it publishes the data in my MQTT server where my ESP32 and Home Assistant get their numbers.

The screen also displays the weather forecast, but you can read more about it here (or don't, it's basically the exact same, but with weather).
